# 🎵 Lyrics Finder + Music Player

A sleek web app that lets users search for song lyrics and listen to the track using YouTube integration. Built with HTML, CSS, and JavaScript.

## 🚀 Features

- 🔍 Search for any song by artist and title
- 🎤 Get full lyrics fetched via API
- ▶️ Embedded YouTube music playback (audio-only)
- 🎧 Clean, responsive UI with animated background
- 🎨 Glassmorphism card design

## 🛠 Tech Stack

- HTML5
- CSS3 (Glassmorphism + Gradient UI)
- JavaScript
- [Lyrics.ovh API](https://lyricsovh.docs.apiary.io/) for lyrics
- YouTube IFrame API for music playback
